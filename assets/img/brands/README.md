## Fuentes / procedencia

- **SimAcademy** — logo oficial del producto. Copia de `assets/logo.png` (canonical original del repo). Versión horizontal, alta resolución.
- **HumanizArte** — socio co-organizador del Diplomado de Actores de Simulación. Copia de `assets/humanizarte-logo.png`. Logo muy chico (136×137); ver notas de uso.
- **EuSim** — afiliación europea de los cursos `eusim1` y `eusim2`. Copia de `assets/eusim-logo.png`. Archivo entregado por EuSim, recibido vía descarga 2026-05-11 (mismo SHA que el archivo ya presente en el repo: `9a2f45fa…`).

## Notas de uso

- **No mover los archivos originales** en `assets/logo.png`, `assets/humanizarte-logo.png`, `assets/eusim-logo.png`. Están referenciados desde múltiples landings (`actores/`, `eusim1/`, `eusim1sep/`, `eusim2/`, `ortopedia/`, `index.html`). Esta carpeta `brands/` es el lugar canónico para **buscar** logos en el futuro; las landings existentes siguen usando las rutas viejas hasta que se migren.
- **HumanizArte (136×137)** es muy chico para hero/print. Usar solo en navbar/footer o badges. Pedir versión vectorial (SVG) o ≥512px a HumanizArte antes de cualquier uso impreso o hero.
- **EuSim (284×170)** también es chico y sin SVG. Suficiente para navbar/badge pequeño en `eusim1/`, `eusim2/`. Para hero o print, pedir SVG al contacto EuSim.
- **SimAcademy (3858×1065)** alta resolución, sirve para cualquier uso. Recordar que el branding SimAcademy usa también tokens de color: verde `#4CAF50` + morado `#7B3F9E` + accent `#D4A35E`, tipografía Montserrat.
- Todos los logos están en PNG con alfa: usar sobre fondo claro idealmente. Para fondos oscuros, validar legibilidad — los tres tienen detalle oscuro que se pierde en oscuro.
